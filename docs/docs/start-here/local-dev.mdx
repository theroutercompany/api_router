## Prereqs

- Go 1.22+
- Node 20+ (for docs site)

## Run the gateway

```bash
go run ./cmd/apigw init --path gateway.yaml
go run ./cmd/apigw run --config gateway.yaml
```

Deep dive:

- Guide: [Run the gateway](../guides/running)
- Reference: [Config](../reference/config), [Environment variables](../reference/env)
- Code: [CLI](../annotated/cmd/apigw/main), [Runtime](../annotated/pkg/gateway/runtime/)

Health endpoints:

- `GET http://localhost:8080/health`
- `GET http://localhost:8080/readyz`

## Run tests

```bash
go test ./...
```

If you want a quick end-to-end sanity check against a running instance:

- Tooling: [Smoke script](../guides/smoke)

## Run the docs site

```bash
cd docs
npm start
```
