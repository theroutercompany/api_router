## Local run

```bash
go run ./cmd/apigw init --path gateway.yaml
go run ./cmd/apigw run --config gateway.yaml
```

Next steps:

- Verify endpoints with the [Smoke script](./smoke)
- Configure ports/upstreams/secrets: [Config](../reference/config), [HTTP config](../reference/config#http), [readiness config](../reference/config#readiness), [Environment variables](../reference/env)

## Hot reload config

```bash
go run ./cmd/apigw run --config gateway.yaml --watch
```
